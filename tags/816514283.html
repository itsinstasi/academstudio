Как выбрать картину для интерьера, чтобы она гармонично вписалась в общий стиль?
По сюжету или цвету?
Если интерьер тематический -то картину нужно выбирать по содержанию. Пусть даже по цвету она будет немного выпадать из контекста. Это не так страшно, как тематико-стилистическое несоответствие.✨
⠀
🌈Но есть универсальное решение — это абстракции, портреты, изображения цветов. С такими произведениями искусства можно опираться только на их палитру.
